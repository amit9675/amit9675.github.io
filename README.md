# Amit Bhandari — Portfolio

A modern, responsive developer portfolio showcasing my work as a **Data Engineer** and **MERN Stack Developer**.

## About Me

I build scalable data pipelines, data platforms, and full-stack web applications. My experience spans data engineering, cloud platforms, databases, and modern JavaScript development.

- **Data Engineering:** Python, SQL, PostgreSQL, PySpark, Databricks, Snowflake
- **Data Concepts:** ETL/ELT, data lineage, normalization, data modeling, and data warehousing
- **Cloud:** AWS and Microsoft Azure
- **MERN Stack:** MongoDB, Express.js, React, and Node.js
- **Problem Solving:** Data structures, algorithms, and 600+ coding problems

## Portfolio Features

- Responsive design for desktop, tablet, and mobile
- Dark theme by default with an optional light theme
- Animated hero, aurora background, particles, and scroll effects
- Projects, technical skills, competencies, and milestones
- GitHub and LeetCode activity
- Downloadable resume and contact links
- Accessible reduced-motion support

## Built With

- React 19
- Vite
- Tailwind CSS 4
- Framer Motion
- React Icons
- React Type Animation

## Run Locally

```bash
git clone https://github.com/amit9675/amit9675.github.io.git
cd amit9675.github.io
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```
